Collections

Collections such as lists and arrays have gotten more extensions to improve your workflow as well. 

using ExtensionsTools;

...

string[] array = new string[] { "this", "is", "a", "test" };
int index = 3;
if (array.IsIndexValid(index))
    return array[index]; 
return ""

Here is a list of all extensions for Lists or arrays

MethodDescription
ShuffleShuffles this collection
IsArrayIdenticalCheck if an array is identical to another array
UnionCreates an union of this collection and another collection
IntersectCreates an intersection of this collection and another collection
DifferenceCreates a difference of this collection and another collection
AddUniqueAdd a value to the collection if it doesn't exist in this collection yet
GetRandomGets a random value from this collection
RemoveElementsRemove all elements equal to this value
IsIndexValidChecks if this index is valid in the collection and returns true if so
ResizeResize this collection
